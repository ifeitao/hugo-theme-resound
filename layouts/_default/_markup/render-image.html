{{- /* Render hook for Markdown images */ -}} {{- $src := .Destination | safeURL -}} {{- $title := .Title -}} {{- $alt := .Text | plainify | htmlUnescape | replaceRE "\\s+" " " | trim " " -}} {{- if not $alt }}{{ $alt = (path.Base $src) }}{{ end -}} {{- $img := .Page.Resources.GetMatch .Destination -}} {{- $width := 0 -}} {{- $height := 0 -}} {{- if $img -}} {{- $width = $img.Width -}} {{- $height = $img.Height -}} {{- end -}}
